Workshop on Real-World Decentralized Cryptography

January 15, 2021
Registration

With all the current interest related to personal data ownership, there is a growing need for a better security, freedom and control for online users, and with this a counter-movement for more decentralized applications where no single entity controls the system has started to grow over the last decade. Blockchains technologies are proposed to address these issues, in the quest to build a decentralized web.

Abstract

The goal of this workshop is to address the challenges that occur when distributed cryptographic protocols are implemented in decentralized set up such as blockchain systems where there is no central entity to coordinate and validate the correct participation in the protocol.

In the workshop we will discuss topics such as (publicly) verifiable secret sharing, distributed key generation, robust secret sharing that have applications in decentralized setups for ZK proofs, threshold signatures, randomness beacons for blockchain technologies.

Although these topics are heavily discussed in the literature, most current solutions lack robust necessary for the real-world usage: many protocols require that any failure result in restarting the process with dropping a failed node or adding/removing nodes and consequently the threshold may need to be changed etc.

The first goal of this workshop is to review the state-of-the-art research and practices. A second goal is to discuss existing challenges in terms of security and performance. The final goal is to trigger new research directions to tackle the challenges. While the main speakers will be invited from academia we will include some industrial researchers from pioneering projects, who are familiar with challenges that appear only when such protocols are deployed.

Besides being a networking opportunity for students, academic and industrial researchers, this combination would not only help academic researchers to get familiar with practical challenges/considerations and existing gaps but also help industrial researchers to get familiar with scientific solutions/considerations and hopefully be the basis for more tight collaboration.
